* cl-scryfall

This is a work in progress api wrapper for scryfall.com. Every API method for cards, rulings and sets is currently implemented

** Example
#+BEGIN_SRC lisp
(ql:quickload :cl-scryfall)
(cl-scryfall:cards-random)
#+END_SRC
